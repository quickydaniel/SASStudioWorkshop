ComboTree jQuery Plugin Demos

Github Home Page

Multi Selection

  comboTree1 = $("#justAnInputBox").comboTree({
    source: SampleJSONData,
    isMultiple: true,
    cascadeSelect: false,
    collapse: true,
    selectAll: true,
  });

Multi Selection With Cascade Option Select

  comboTree3 = $("#justAnInputBox1").comboTree({
    source: SampleJSONData,
    isMultiple: true,
    cascadeSelect: true,
    collapse: false,
  });

Single Selection

  comboTree2 = $("#justAnotherInputBox").comboTree({
    source: SampleJSONData,
    isMultiple: false,
  });

Default Settings

  const defaults = {
    source: [],
    isMultiple: false,
    cascadeSelect: false,
    selected: [],
    collapse: false,
    selectAll: false,
    animationTime: 200,
